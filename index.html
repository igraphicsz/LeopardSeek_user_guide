    LeopardSeek

    本页介绍了 LeopardSeek 的基本情况

    LeopardSeek 概括

    LeopardSeek 是由波普尔技术开发的针对中国期货的量化交易工具，为用户提供可配置参数的，自由度极高的量化交易策略编写环境，支持指定时间段回测，模拟盘以及实盘量化交易等丰富的功能。

    用户输入

    LeopardSeek 至少要求你提供一套交易策略。给定行情数据后，你的策略需以委托单（Order）的形式输出交易信号。通常仅需满足这一条件，你即可对策略进行回测；但如果想要模拟盘/实盘交易，还需在 config.yaml 配置文件中填写你的交易账户与密码。

    项目结构

    your_trading_project/
├── start_virtual_trade.py          # 启动实盘交易
├── start_live_trade.py             # 启动模拟盘交易
├── start_backtest.py               # 启动回测
├── config.json                     # 全局配置
├── config 
│   ├── strategy1_config.yaml       # 策略1配置文件
│   └── strategy2_config.yaml       # 策略2配置文件
├── strategies
│   ├── strategy1.py                # 策略1，定义了包含策略1逻辑的Strategy
│   └── strategy2.py                # 策略2，定义了包含策略2逻辑的Strategy
├── strategy_logs
│   ├── ao2605.log                  # ao2605的日志
│   ├── m2605.log                   # m2605的日志
