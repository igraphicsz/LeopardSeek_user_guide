    下单

    本页面介绍如何下单以及订单不同参数的含义及用途

    生成订单

    交易订单可以通过实例化 Order 类实现，不同委托单类型及其对应的必填项要求，详见下文说明。

    订单参数

    LeopardSeek 封装好了便捷的订单参数方便用户调用，仅需要在策略文件夹开头引用需要的 python枚举 即可。

    from autotrader.comms.Enum import Direction, Offset, OrderType

    Direction, Offset, OrderType 均为 python枚举类（Enum）, 封装了生成交易订单所需要的参数以供用户生成便捷，直观地订单。他们的定义如下：

    class Direction(Enum):
    Long = 1    # 多
    Short = -1  # 空

class Offset(Enum):
    Open = 1    # 开仓
    Close = 2   # 平仓/平昨 除了上期所/能源中心外，不区分平今平昨

class OrderType(Enum):
    Limit = 1   # 限价单
    Market = 2  # 市价单

    以一个市价单为例，以下是一个市价开10手多仓的订单示例：

    Order(instrument=self.instrument, # 产品代码
        direction=Direction.Long, # 多/空
        offset=Offset.Open, # 开/平
        order_type=OrderType.Market, # 以市价成交
        price=current_price, # 为避免报错，即使市价成交请依然传入price参数
        size=10 # 交易手数
        )

    下单示例

    LeopardSeek 已封装中国期货交易下单所需的 API，仅需将订单通过 generate_signal 方法返回即可。

    LeopardSeek 接受返回单个订单或者一系列订单（以订单列表的形式）:

    返回单个订单（开多仓）

    # generate_signal...

# 首先，通过get_candles方法获取当前价格
price_data = self.broker.get_candles(instrument=self.instrument,
                                          granularity=CandlesPeriod.MIN_3,
                                          count=count_3min,
                                          end_time=dt
                                          )
# 下文中的下单操作均有使用current_price(当前价格)                                          
current_price = price_data['close'].iloc[-1]

# 返回单个开多仓的订单
order = Order(instrument=self.instrument, # 产品代码
        direction=Direction.Long, # 多
        offset=Offset.Open, # 开
        order_type=OrderType.Market, # 以市价成交
        price=current_price, # 为避免报错，即使市价成交请依然传入price参数
        size=10 
        )

return order

    返回订单列表（平多开空）

    # generate_signal...

order1 = Order(instrument=self.instrument, # 产品代码
        direction=Direction.Long, # 多
        offset=Offset.Close, # 平
        order_type=OrderType.Market, # 以市价成交
        price=current_price, # 为避免报错，即使市价成交请依然传入price参数
        size=10 # 交易手数
        )

order2 = Order(instrument=self.instrument, # 产品代码
        direction=Direction.Short, # 空
        offset=Offset.Open, # 开
        order_type=OrderType.Market, # 以市价成交
        price=current_price, # 为避免报错，即使市价成交请依然传入price参数
        size=10 # 交易手数
        )

orders = [order1, order2]

return orders
