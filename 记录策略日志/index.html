    记录日志

    LeopardSeek为策略交易的每个资产代码都内置了独立的日志，便于使用者记录策略执行过程中信号的变化情况。

    from logging import Logger # 引入python内置日志模组

# ... generate signal

      # 通过generate_features生成趋势信号
      signal = self.generate_features(dt)

      # 通过日志记录交易信号
      self.logger.info(f'交易信号{signal}')

    以资产代码 'm2605' 为例，在策略文件中调用完成后，可在根目录 strategy_logs 文件夹下 m2605.log 文件中查看交易信号的变化情况

    2026-02-24 10:31:05,574 |     INFO |    m2605 | 交易信号-1 (macd.py:80)
2026-02-24 10:32:13,994 |     INFO |    m2605 | 交易信号-1 (macd.py:80)
2026-02-24 10:33:06,130 |     INFO |    m2605 | 交易信号0  (macd.py:80)
