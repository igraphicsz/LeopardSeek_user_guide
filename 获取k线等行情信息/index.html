    获取市场行情信息

    本页面介绍如何借助Leopardseek获取k线数据及更多市场行情信息

    获取市场行情信息

    获取支持的k线周期

    LeopardSeek 提供了两个方法用于获取k线数据及更多市场行情信息，为 lzdc_broker 模块下 Broker 类中 get_candles 和 get_full_market_data 方法获取行情信息。

    为使用 get_candles 和 get_full_market_data 方法获取行情信息，需要在策略（.py）文件的开头部分引用 CandlesPeriod 来获取 LeopardSeek 支持的k线周期。

    from autotrader.comms.Enum import CandlesPeriod

    其中 CandlesPeriod 为 python枚举类（Enum）定义如下

    # Leopardseek目前所支持获取的k线周期
class CandlesPeriod(Enum):
    MIN_1 = '1min'
    MIN_3 = '3min'
    MIN_5 = '5min'
    MIN_10 = '10min'
    MIN_15 = '15min'
    MIN_30 = '30min'
    H_1 = '1h'

    获取行情数据

    在策略的 generate_signal 方法中，调用 get_candles 和 get_full_market_data 获取市场行情信息

    # 获取k线数据，仅包括OHLCV
data_candles = self.broker.get_candles(instrument=self.instrument,
                                          granularity=CandlesPeriod.MIN_10,
                                          count=10,
                                          end_time=dt
                                          )

# 获取完整市场行情数据
data_full = self.broker.get_full_market_data(instrument=self.instrument,
                                          granularity=CandlesPeriod.MIN_10,
                                          count=10,
                                          end_time=dt
                                          )

# ... 略过了策略的 __init__ 方法与 generate_signal 前半部分

data = self.broker.get_candles(instrument=self.instrument,
                                          granularity=CandlesPeriod.MIN_3,
                                          count=200,
                                          end_time=dt
                                          )
# 获取每根k线的收仓价用于计算均线，具体列名见上述打印的Dataframe
close_prices = data_3min['close']
